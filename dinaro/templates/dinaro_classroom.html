Dinaro for Classrooms

Run a tiny classroom economy. Students earn dinaro for tasks, save and spend it, and vote on how to use a shared class fund - real economics, zero real money.

The Treasury

A shared class fund with a goal and a teacher match. Students are billed their fair share (taxes), and they vote on how the fund is spent - a hands-on lesson in taxation, public goods, and collective choice.

Class analytics

A teacher dashboard with a leaderboard, average balance, tasks completed today, and completion rate - see the whole class at a glance. Optional grade mode, interest, and tax-rate settings.

How students join

Share your class code (or QR). Students log in with the code and a PIN, earn dinaro for tasks you set, and only ever see their own balance.
