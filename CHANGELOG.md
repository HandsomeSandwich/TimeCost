# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.3.0] - 2026-02-25

### Added
- **Classroom Mode**: Toggle in Parent Settings to switch terminology to Teacher/Student/Task.
- **Wealth Trend Charts**: Visual 7-day balance history for all children on the Parent Dashboard.
- **Recurring Subscriptions**: Added "Expense" chore type for automated daily deductions (e.g., tech fees).
- **Child History Expansion**: Children can now view their full transaction ledger.
- **Parental Goal Control**: Parents can now edit or delete children's savings goals.
- **Overtime in Visual Mode**: Simplified overtime logging for younger kids.

### Changed
- Database optimization: Pruned legacy migration code and streamlined schema updates.
- Dashboard improvements: Added "To-Do" progress indicators for parents.
